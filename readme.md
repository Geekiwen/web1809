1.Git命令
    1.初始化命令：
        git config --global user.name "Your Name"
        git config --global user.email "email@example.com"
    2.初始化仓库
        git init
    3.增加版本管理
        git add filename
        git commit -m '注释'
    4.仓库状态
        git status
        git diff
    5.版本回退
        git log
        git reset --hard HEAD^
        git reset --hard 1094a
